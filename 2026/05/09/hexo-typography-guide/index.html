常用命令

命令 作用
hexo server 启动本地预览，默认 http://localhost:4000
hexo new "文章标题" 在 source/_posts/ 创建新文章
hexo generate 生成静态文件到 public/ 目录
hexo deploy 部署到远程服务器（需配置 deploy 段）
hexo clean 清除缓存和静态文件

    # http模块

    # 什么是客户端 什么是服务端

    在网络节点中，负责消耗资源，叫客户端；负责对外提供网络资源的，叫服务器

    # 什么是http模块

    nodeJs官方提供的，用来创建服务器的模块，通过http的createServe方法，就能把电脑变成web服务器，提供web资源服务

    举了一个例子 php 等等服务器

    我们自己也可以手撸一个网站

    # http-相关的概念

    # IP地址

    点分十进制

    # 域名

    字符型的地址方案

    给IP起一个别名

    对应关系放在一种叫做 域名服务器 的电脑中

    域名服务器就是提供IP地址和域名之间的的转换服务的服务器

    127.0.0.1 对应的域名是 localhost

    # 端口号

    通过唯一的端口号，可以被准确的交给对应的web服务进行处理

    每个端口号只能被一个web服务占用

    80端口可以被省略

    # 创建最基本的web服务器

    # 创建web服务器的基本步骤

    导入 创建 绑定request，监听客户端的请求 启动服务器

    和go一样的创建端口都非常的快！！！

    # req请求对象

    # res响应对象

    访问与服务器相关的数据或属性

    # 模块作用域和module对象

    # 什么是模块作用域

    自定义模块中定义的变量、方法，只能在当前模块内被访问

    这种模块级别的访问限制，叫做模块作用域

    好处 全局变量污染的问题

    # 向外共享模块作用域中的成员

    在每个 .js 自定义模块中都有一个module对象，它里面存储了和当前模块有关的信息

    什么是module呀，指向一个空对象

    # exports 和 module.exports 的使用误区

    反正用module.exports就对了

    否则不用 module.exports = {} 因为这是指向新对象

    而 exports = {} 的情况下，只要有module.exports，那就是后者

    只要不用赋值，就是好兄弟

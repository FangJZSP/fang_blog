    年度总结｜2023

    Fang 随想

    # 2023年度总结

    # 前言

    看别人洋洋洒洒的写了一堆年终总结，感觉自己也该写一点，放着。 然后展望一下2024年。

    # 2023经历

    2023年初，知道自己考研失败了，立马开始找工作，然后就是开始学习java，说自己一定要转互联网，写代码。这样未来和女朋友说不定有更多的话题和交集。

    然后base也可以在武汉，那是再好不过了。两个人离得近。

    开始学java真的很有意思，因为有了计算机知识的基础，学起java来兴致勃勃。

    一路上，就是各种过，敢于写基础的题目。

    后面学到框架，学到项目就开始跟着老师一点点的敲，感觉进步是有的，但更多的是对于八股文的记忆和理解。

    这段时间，刷了java的很多视频，看到了很多人的代码，终于一步步的对于java的增删改查有了眉目。

    懂了数据库，懂了ssm框架，懂了Springboot，懂了缓存等等东西。

    虽然最后没有微服务，因为自己的电脑已经带不动更难的项目了。

    说实在的，内心也很急躁。每天也是高强度的网上冲浪。看大家的进度是怎么样的。

    手里又没钱，在家里心里压力也大。

    但是努力，才能有回报。

    # 当时的备忘录

    苦练潜修功自成 不惮劳苦志凌云 鼎炉丹转恋纯火 攀桂擎鳌乐意融

    距离23考研已经过去了357天了，很感慨时间过的很快，可能再过几天就是24届考研了。这一年也没想到自己能挺过来，我还记得我写了一个23年生存指南，不知道现在还在不在哈哈。总结来说就是超出预期的吧。

    23年初，考研失败，于是乎开始准备学习java。然后22年尾，面试了一家培训机构，当晚辅英语老师，还好有一点底子托底，不然都不知道怎么挣第一桶金。

    这份工作一直到4月份，每个月有几百一千的收入，终于可以不向爸妈伸手要钱生活了，当然吃喝还是在家里。当晚辅老师苦不苦，有点吧。刚开始去的时候都不太适应。最后其实也没什么活，无非就是给高三复读生讲讲单词语法，各种题型怎么解答，大家的基础也不是很好，像我这种也可以应付应付，不过同时我还是把高中的语法书还读了一本，这也是种进步吧。如果单想着来去各1小时，上班4小时，6点去，11点回，坐完地铁，骑共享单车，就这样回家，工资100块钱，那这段时间还是有点难的，哈哈。看到那些同学的时候我都还有些感慨，学生时代大概就是从那个时候离我而去吧。

    不过这一段时间，终于让我开始接触社会，开始感觉自己有能力挣钱，自己也能花钱，不再有所谓父母给你的钱是困住你的桎梏了。

    23考研失败，不过是大学的一个缩影罢了。一个从丧失学习动力的人从泥潭中挣扎出来的救赎。

    然后6月份很着急了，开始投简历，面了几家不靠谱的公司。

    最后收了两个offer。 第一个是自己找到的，小公司，试用4k转正5k，然后上班时间忘了，反正应该会做很多杂事。

    第二个offer是女朋友帮我找到的，然后我去面试了，通过了试用期，最后转正了，忙忙碌碌完成了考核。再然后完成小需求，到完成一个版本的上线。再到现在的看数据，进行项目的维护。要说什么时候代码能力开始突飞猛进的。那肯定是，开始真正的自己根据ui开始写需求，开始写实现。一步步的从ui转换为代码的时候，代码能力开始突飞猛进，愿意去看更难的代码，去学习里面的思想。

    最后终于在年尾，过上了一个幸福的年。

    # 2023收获

    1. 收获了家人的爱，小丫的爱
    2. 情绪稳定
    3. 买了两大件 手机+电脑
    4. 工作顺利，有了一点积蓄

    # 2024展望

    # 工作学习方面

    1. 算法一百题
    2. 精进dart-flutter/java-Springboot，学习nodejs
    3. 维护一个自己的开源项目

    # 生活方面

    1. 给家庭购置一到两件电器
    2. 给女朋友更多的关心和爱

    # 最终寄语

    乘风破浪会有时，直挂云帆济沧海。
